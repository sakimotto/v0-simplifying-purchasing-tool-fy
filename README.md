# Zervi Sourcing App

A comprehensive procurement platform for garment and textile businesses. Streamline supplier management, order tracking, sample evaluation, RFQ workflows, and sourcing analytics — all in one place.

## Features

- **Supplier Management** — directory with detailed profiles, comparison tools, verification tracking
- **Order Management** — create and track purchase orders with delivery monitoring
- **Sample Management** — request, track, and evaluate physical samples
- **RFQ System** — create and distribute requests for quotes, compare responses
- **Document Management** — centralized repository with categorization
- **Analytics Dashboard** — spend analysis, supplier performance, category trends

## Tech Stack

- Next.js 15 (App Router) + React 19 + TypeScript
- Tailwind CSS + shadcn/ui components
- pnpm package manager

## Getting Started

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000).

## Project Status

Early development — UI prototype phase. Backend, database, and authentication are planned for upcoming phases.
